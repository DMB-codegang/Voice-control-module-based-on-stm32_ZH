# Voice-control-module-based-on-stm32_ZH
基于STM32的语音控制模块-中文

本仓库会提供一个以STM32为核心板的语音控制模块的所有资料  
持续更新  
  
  
———————————————主要材料———————————————  
1.STM32F103C8T6开发版（40P）  
2.LD3320语音输入（12P）  
3.SYN6288语音输出（5P）  
4.底板（打样板，嘉立创可以白嫖，文件见后文）
